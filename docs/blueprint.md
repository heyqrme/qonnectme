# **App Name**: Qonnectme

## Core Features:

- Unique QR Code Generation: User receives a unique QR code upon registration, serving as their digital identifier.
- Customizable User Profiles: Personalized user profiles with options to add personal information, music, photos, and videos.
- Friend Request System: Enables users to send and accept friend requests, fostering a connected social network.
- Multimedia Sharing: Platform for users to upload and share multimedia content like music, photos, and videos on their profiles.
- QR Code Merchandise Store: Online store integrated into the app, allowing users to purchase items featuring their unique QR codes. This will use dummy payment.
- Account Creation for Non-Subscribers: Enables users without a paid subscription to create an account and connect with others within the network.
- AI-Powered Theme Suggestions: Suggest personalized profile themes using AI tool based on the user's uploaded music and media.

## Style Guidelines:

- Primary color: Soft purple (#A080DD) to create a soothing and modern feel.
- Background color: Dark mode background (#121212) to reduce eye strain and enhance the vibrancy of UI elements.
- Accent color: Soft pink (#FF69B4) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern and neutral appearance, which makes it highly versatile for both headlines and body text.
- Use minimalist, custom icons with a line art style, colored in soft pink.
- Implement a card-based layout to organize user profiles, posts, and store items effectively.
- Subtle transitions and animations to enhance user engagement, such as sliding transitions or QR code scanning feedback.
